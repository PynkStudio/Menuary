# Feature

Specifiche delle feature della piattaforma. Ogni feature ha un proprio documento basato sul [[adr-template|template feature]].

## Cosa inserire qui

- Specifiche funzionali di ogni feature
- Requisiti e criteri di accettazione
- Stato di implementazione
- Note su feature-flag e configurazione per tenant

## Convenzione

Usa il template [[feature-template]] per ogni nuova feature.
