# Troubleshooting — KB

Risoluzione dei problemi più comuni, organizzata per **sintomo** percepito dall'utente.

- Template: [[kb-troubleshooting-template]]
- Sezioni standard: `# Sintomo`, `# Cause possibili`, `# Verifiche`, `# Soluzione`, `# Quando aprire un ticket`.
- La sezione `# Quando aprire un ticket` collega alla [[escalation-policy]].

Nomina i file con il sintomo (es. `pagamento-non-accettato.md`, `prenotazione-non-arriva.md`).
