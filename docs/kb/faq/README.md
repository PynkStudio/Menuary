# FAQ — KB

Domande frequenti, **una per file**, formulate nel linguaggio reale dell'utente.

- Template: [[kb-faq-template]]
- Sezioni standard: `# Domanda`, `# Risposta`, `# Note`, `# Documentazione correlata`.
- La risposta deve essere autosufficiente: l'assistente deve poterla restituire senza ulteriore contesto.

Nomina i file con lo slug della domanda (es. `come-cambio-orari-apertura.md`).
