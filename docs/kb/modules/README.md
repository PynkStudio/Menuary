# Moduli — KB

Una scheda per ogni modulo della piattaforma, pensata per spiegare a un `tenant_admin` cosa fa il modulo e come usarlo.

- Template: [[kb-module-template]]
- Sezioni standard: `# Descrizione`, `# Funzionalità`, `# Permessi richiesti`, `# Flussi operativi`, `# Limitazioni`, `# FAQ correlate`.
- Riferimento all'elenco moduli del codice: [[moduli-piattaforma]].

Nomina i file con lo slug del modulo (es. `prenotazioni.md`, `menu-online.md`).
