# Glossario — KB

Termini e definizioni della piattaforma, per allineare il linguaggio tra utenti, documentazione e assistente AI.

- Una voce per termine, oppure un unico file alfabetico se il numero è contenuto.
- Frontmatter minimo consigliato: `title`, `tags`, `last_updated`, `owner`.
- Definizioni brevi e autosufficienti (utili come chunk RAG indipendenti).

Termini candidati ricorrenti nella piattaforma: `tenant`, `verticale`, `modulo`, `feature-flag`, `tenant_admin`, `preview/demo`, `gestione`, `escalation`, `ticket`. Definiscili in base alla documentazione esistente ([[panoramica-tecnica]], [[moduli-piattaforma]]) — senza inventare significati non confermati.
